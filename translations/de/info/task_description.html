M. C. Escher war ein niederländischer Grafiker, der unglaubliche Illustrationen schuf, die von der Mathematik inspiriert waren. Zum Beispiel füllte er die Leinwand mit selbstähnlichen Objekten, deren Konturen sich ineinander fügten und so sehr beeindruckende geometric symmetries schufen.

Ein sehr einfaches Beispiel für diese Idee sehen Sie in der folgenden Abbildung, die ein Objekt zeigt, das ein orthogonales Profil ist, das durch eine Folge von natürlichen Zahlen definiert ist, die die Folge von Höhen darstellen. Wir können eine Kopie des Objekts nehmen, sie um 180 Grad drehen und sie perfekt in das ursprüngliche Objekt einpassen, so dass ein Rechteck entsteht.

Allgemeiner ausgedrückt: Wenn eine Folge von N natürlichen Zahlen, die die Abfolge der Höhen repräsentieren,A1,A2,A3, ...,AN-2,AN-1,AN ist, wird das definierte Profil als Escher-Profil bezeichnet, wenn A1+AN gleich A2+AN-1 gleich A3+AN-2 ist, und so weiter. In diesem Problem wird Ihnen die Folge von Höhen gegeben, die das Profil definieren und Ihr Programm muss entscheiden, ob das Profil ein Escher-Profil ist oder nicht.
